# Virtual Museum: Santo Domingo Chapel (Tunja)

A Unity-based virtual museum featuring an immersive virtual reality tour of the Santo Domingo Chapel in Tunja. The project integrates the Meta VR SDK to deliver a guided, interactive, room-scale experience.

## Objectives
- Digitally preserve the cultural and architectural heritage of the Santo Domingo Chapel.
- Provide an accessible and immersive learning environment using VR technologies.
- Strengthen student competencies in Unity development, VR interaction design, and immersive environments.

## Key Features
- High-fidelity 3D reconstruction of the chapel.
- Guided VR tour with informative hotspots.
- Meta VR SDK integration for interaction, locomotion, and hand tracking.
- Optimized performance for standalone Meta headsets.

## Technology Stack
- **Engine:** Unity
- **VR SDK:** Meta XR / Meta Interaction SDK
- **Platform:** Meta Quest family
- **Assets:** Custom 3D models, photogrammetry elements, Unity HDRP/URP (project-dependent)

## Project Team
Developed by students of the **Videogames and Metaverse Research Group** at **Universidad Santo Tomás**.

## Repository Structure
- `Assets/` – Unity project assets (models, textures, scripts)
- `ProjectSettings/` – Unity project configuration
- `Documentation/` – Reference material, design docs
- `Builds/` – Optional VR builds (if versioned)

## Setup Instructions
1. Install Unity (recommended LTS version for the project).
2. Clone the repository.
3. Open the project with Unity Hub.
4. Install required Meta XR packages through Unity Package Manager.
5. Set build target to Android and configure Meta Quest settings.
